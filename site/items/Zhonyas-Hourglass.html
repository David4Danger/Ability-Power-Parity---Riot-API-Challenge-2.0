Summary: Zhonya's Hourglass recieved the 120 AP to 100 AP adjustment like Luden's Echo did, along with the 300g discount. The only difference here is the added utility Luden's Echo got, while Zhonya's just got appropriately scaled down. As the number above dictate, this was not very impactful on winrate or popularity. As long as Zhonya's has decent stats, it will always be a staple due to it's active. New patch, same old Zhonya's.

Rumble built Zhonya's 62.99% of the time in patch 5.11.
Rumble built Zhonya's 66.80% of the time in patch 5.14.
Rumble built Zhonya's 3.81% more after it was updated.

Zhonya's Hourglass got a bit cheaper with a bit less ability power tacked on to it, so it seems Rumble players were able to get their hands on it a bit more often with the cheaper price. This item is heading towards staple-popularity on Rumble.

Kennen built Zhonya's 74.91% of the time in patch 5.11.
Kennen built Zhonya's 78.53% of the time in patch 5.14.
Kennen built Zhonya's 3.62% more after it was updated.

With the cheaper pricetag, Kennen players found themselves buying Zhonya's hourglass even more often than before. The only real downside to the item for a Kennen was the hefty pricetag, as the real objective here is that amazing passive.

Morgana built Zhonya's 63.57% of the time in patch 5.11.
Morgana built Zhonya's 66.68% of the time in patch 5.14.
Morgana built Zhonya's 3.11% more after it was updated.

Morgana players also found themselves with a few extra Zhonya's, as the price decrease definitely fit in some budgets that it previously did not. Whether Morgana is mid or support, Zhonya's fits her playstyle very well, don't expect this number to drop any time soon.

Heimerdinger built Zhonya's 78.75% of the time in patch 5.11.
Heimerdinger built Zhonya's 79.19% of the time in patch 5.14.
Heimerdinger built Zhonya's 0.44% more after it was updated.

It isn't surprising that Heimerdinger players are still building Zhonya's hourglass. It doesn't matter how fed he is, in teamfight the donger will always be everyones first target (thanks, qtpie).

Soraka built Zhonya's 3.10% of the time in patch 5.11.
Soraka built Zhonya's 2.56% of the time in patch 5.14.
Soraka built Zhonya's -0.54% less after it was updated.

Just to be clear, this likely has nothing to do with the changes to Zhonya's Hourglass, no, this is about an incredible injustice. Riot "fixed" the "bug" that caused Soraka to have her unique colour when using Zhonya's hourglass.

There can be no doubt that the massive drop in Soraka Zhonya's population is due to this maltreatment.
