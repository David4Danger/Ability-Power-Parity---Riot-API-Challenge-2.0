Summary: Morellonomicon recieved the same treatment as Athene's, making it slightly friendlier in the early game so it could compete with some other select AP items. Like Athene's, an increase in winrate followed but popularity seemingly dipped. Infact, the amount of Morellonomicons built per game had nearly the exact same percentage drop as Athene's. Coincidence? I THINK NOT.

"ARE THESE CHOICES MEANINGFUL? You tell us." Ahri players everywhere rejoice thanks to some crisp changes, even if it is just a book.

Lulu built Morellonomicon 30.07% of the time in patch 5.11.
Lulu built Morellonomicon 30.23% of the time in patch 5.13.
Lulu built Morellonomicon 0.17% more after it was updated.

The changes to Morellonomicon were meant to promote buying the item more often since it was friendlier to build early. It is clear this change was of no concern to Lulu players, who are picking it up just as much as they were before. The relatively low overall build rate is due to Lulu's popularity as a support champion.

Ahri built Morellonomicon 64.45% of the time in patch 5.11.
Ahri built Morellonomicon 66.69% of the time in patch 5.14.
Ahri built Morellonomicon 2.24% more after it was updated.

The changes to Morellonomicon seem to have made it slightly more favourable with Ahri players. Hopefully this small change in popularity is related to Ahri players opting for Morellonomicon earlier instead of the Needlessly Large Rod first back purchase.

LeBlanc built Morellonomicon 67.79% of the time in patch 5.11.
LeBlanc built Morellonomicon 66.25% of the time in patch 5.14.
LeBlanc built Morellonomicon -1.54% less after is was updated.

Unlike fellow burst mage Ahri, LeBlanc players didn't take to the updated Morellonomicon. This is well within expected variance but the real point here is that none of these Morellonomicon building champions saw a real increase in how often they built this item - proving the changes to have not much of an impact. LeBlanc is no exception to this.
