With the slow stacking changes in patch 5.13, Rylai's needed to be adjusted as to not risk breaking the game. The changes applied to Rylai's still make it stronger than previous with a new build path and a stronger passive - if you use it correctly. As expected, Rylai's winrate increased to the highest of it's breed, making it the top winrate of any AP item changed in patch 5.13 (tied with Nashor's). All in all Rylai's became much stronger and as a result, a bigger temptation for ability power users to get some variety in their build.

"Truth be told, this thing's going to be pretty strong." The patch notes hit this one spot on, hopefully with it being such a great item, it's popularity reflects this in the coming patches.

Rumble built Rylai's 31.88% of the time in patch 5.11.
Rumble built Rylai's 33.84% of the time in patch 5.14.
Rumble built Rylai's 1.96% of the time in patch 5.14.

The changes to Rylai's and slow stacking might be a bit confusing on the surface but definitely still work well with Rumble's kit. Rumble players build Rylai's 2% more as of patch 5.14, but in the future it would make sense for that number to continually grow as players realize how great an item Rylai's is on Rumble along with the new slow stacking changes.

Before it was updated, Rylai's was very unpopular on the Korean server compared to the global average. Changing the item and how slow stacking functions did not help this whatsoever. Korea went from 0.11 Rylai's per game to 0.12 Rylai's per game, while the global average went from 0.24 Rylai's per game to 0.32 Rylai's per game.

Someone go tell the Korean server its okay, the item's winrate went up to 58%!
